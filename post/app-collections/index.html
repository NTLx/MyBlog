VSCode

Visual Studio Code 简称 VSCode。不是 IDE 的全能文本编辑器，Microsoft 出品的神器，点此下载。

其优点太多，你只需要知道它是最先进的文本编辑器就好了。另外，再强调一点，它不是 IDE，和 PyCharm 等软件有区别。

  • Git 集成（可视化操作 Git）
  • 强大的自定义能力
  • 强大的插件功能
  • 强大的兼容性
  • 强大的社区支持
  • 强大的官方支持

最新特色：可直连服务器进行开发

更多特性请查看官网
